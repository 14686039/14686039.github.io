Mac下SSH终端证书方式到远程服务器

选择证书进行登录，便捷，安全；
我这里也只是一个演示；我在MAC系统下需要远程到其他Linux服务器；

  • 首先，再本地MAC系统下cd ~/.ssh/
  • 使用 ssh-keygen -t rsa 创建公钥和私钥，

    公钥：id_rsa.pub
    私钥：id_rsa
    将公钥文件上传到远程服务器用户主目录下的.ssh文件下（如果没有就新建一个）
  • 登陆到远程服务器，进入.ssh目录 cd ~/.ssh
    执行命令 cat id_rsa.pub >> authorized_keys
    再执行命令 rm id_rsa.pub （删除之前上传的公钥）
    退出远程服务器
  • 在Mac终端下执行登陆命令 ssh username@server；即可免密码登录到远程服务器
